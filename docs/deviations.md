# Deviations Log

## Purpose

This document records **intentional deviations** from the canonical Oqtane AI Playbook.

Deviations are **rare**, **explicit**, and **justified**.

Anything not recorded here is assumed to follow the canonical rules.

---

## Rules for Deviations

A deviation MUST:

- Be explicitly documented here
- Include a clear justification
- State whether it is temporary or permanent
- Be reviewed during code review

Undocumented deviations are considered **defects**.
